# QRQLL 

这是一个轻量级、跨平台的本地 Mock 服务测试工具。内置了生产级的 Waitress Web 引擎。

本项目同时提供 **GUI（图形界面版）** 和 **CLI（命令行版）** 两个版本，满足不同场景的使用需求。

## ✨ 核心特性

- 🌐 **全屏网页内嵌**：支持灵活配置外部 URL，横/竖屏 CSS 彻底分离，完美自适应铺满客户端屏幕。
- 🔍 **自由缩放**：支持对目标网页进行 25% ~ 150% 的按比例缩放，实时生效。
- 📂 **本地资源管理**：自动读取 `resources` 目录，并生成资源分发接口。
- 🔴 **防误触快捷关闭**：支持注入“关闭程序”特殊作业，在客户端点击二次确认后即可远程退出服务端。
- 📝 **支持 JSON 导入导出**：支持一键备份和恢复你的上网配置。
